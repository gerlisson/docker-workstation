# Docker Workstation 
![CI status](docs/images/docker-whale.svg)

Docker Workstation is a ready to use workstation with nodejs, mongoldb, apache and more..

## Features

* None

## Requirements
* Docker and Docker Compose

## Installation

On project root folder use:

`$ docker-compose up -d --build`
